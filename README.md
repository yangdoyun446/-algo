# 소나기
## 소중한 나의 PS일기
알고리즘 공부를 위해 BOJ의 단계별로 풀어보기를 밀기 위한 ,, <br>
다음으로 넘어가는 기준 : 그 알고리즘의 스탠다드 문제 난이도+3의 문제를 보고 왜 그 알고리즘을 썼는지 이해할 정도 <br>
WA를 한번이라도 받은 문제는 : 이슈에 올리고 3일뒤 다시 확인 <br>
